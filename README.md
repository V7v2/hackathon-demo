# Hydro-Guard AI | Farmspherica Innovation
A smart, predictive maintenance dashboard for hydroponic systems.

## Features
- **Real-time Monitoring:** Visualizes pH and nutrient data.
- **AI-Driven Insights:** Simulates predictive maintenance alerts.
- **Responsive UI:** Built with Tailwind CSS and Chart.js.

## Tech Stack
- Frontend: HTML5, JavaScript, Tailwind CSS
- Data Visualization: Chart.js
- Deployment: Vercel
